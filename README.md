#bigdata-customer-analytics-spark
Đây là đồ án môn học Ứng dụng Phân tích Dữ liệu Lớn (Big Data), tập trung xây dựng một End-to-End Pipeline hoàn chỉnh để phân tích dữ liệu thương mại điện tử Olist. Hệ thống sử dụng Apache Spark MLlib để xử lý dữ liệu lớn, xây dựng các mô hình dự đoán hành vi khách hàng, phân khúc thị trường, và cung cấp các khuyến nghị sản phẩm thông minh.
Mục tiêu chính:
Xây dựng quy trình xử lý dữ liệu lớn (Big Data) bằng Spark Pipeline hoàn chỉnh (bao gồm tiền xử lý, mã hóa, vector hóa).
Thực nghiệm và so sánh hiệu quả của 5 nhóm thuật toán Spark MLlib (Classification, Regression, Clustering, Recommendation, Frequent Pattern Mining).
Triển khai kết quả mô hình thông qua giao diện trực quan bằng Gradio, hỗ trợ doanh nghiệp đưa ra quyết định dựa trên dữ liệu.
